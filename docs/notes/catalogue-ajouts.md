*27 juin 2026 · toutes les mécaniques proposées, dédupliquées, en format fiche. Tableau de synthèse : `../IDEES.md`. Sessions détaillées : `2026-06-27_brainstorm-A/B/C.md`.*

## A. Le secret & les objectifs

**Objectifs / Mandats secrets** *(Dead of Winter, Cosmic Encounter, Coup)*
- **Description** : chaque joueur reçoit en début de partie un « Programme » caché (ex. rallier tel type d'électeurs, dénoncer un rival X fois, financer un montant, faire qu'un voisin ne dépasse jamais 15…). Accompli → bonus (voix, carte spéciale, immunité). Révélé à la fin ; le vainqueur peut être recalculé.
- **Ressort addictif** : fil rouge personnel + paranoïa sociale (« pourquoi joue-t-il propre ? »). On rejoue pour tester un nouveau mandat et percer celui des autres.
- **Remplace / restructure** : module optionnel ; peut atténuer le « premier au seuil » brut ; peut remplacer la *raison d'être* des synergies molles.
- **Coût / équilibrage** : faible matériel (cartes Objectif), calibrage délicat (bonus +2 à +4 max). Risque moyen.
- **Ton** : fort (promesses absurdes/cyniques).
- **Compat. maths** : ⚠️ change la condition de victoire. **Vu dans : A, B.**

**La Une du Journal (objectif public tournant)** *(7 Wonders)*
- **Description** : une carte « Une » publique par manche donne un bonus conditionnel (« +2 voix à qui a le plus de cartes sales Presse cette manche » / « le plus propre +3 »). Elle tourne, créant des courses secondaires.
- **Ressort addictif** : la stratégie optimale change chaque manche → pivots, relances.
- **Remplace / restructure** : ajout (course secondaire), ne supprime rien.
- **Coût / équilibrage** : faible. Risque faible.
- **Ton** : ✅✅✅ (le cycle médiatique).
- **Compat. maths** : ⚠️ ajoute des voix. **Vu dans : A.**

## B. Le rythme & le climax de fin

**Soir d'Élection / Sondage de Sortie des Urnes (manche de scrutin)** *(Libertalia, Sheriff, near-miss King of Tokyo)*
- **Description** : périodiquement (toutes les 3 manches) ou au seuil de tension (S−3 voix), un bloc « indécis » fort est mis aux **enchères secrètes** / une **révélation simultanée** a lieu. Le plus offrant (ou l'icône média la plus forte) rafle le bloc / réorganise le marché / force le meneur à piocher une corruption. L'argent misé par les perdants est en partie perdu.
- **Ressort addictif** : pic dramatique (« tout le monde retient son souffle ») + effet *near-miss* + regret (« j'aurais dû miser plus ») = relance immédiate.
- **Remplace / restructure** : ajoute un rythme/climax (3-4 pics par partie), ne supprime rien.
- **Coût / équilibrage** : faible à moyen (un repère sur la piste suffit pour la version sondage). Risque moyen (kingmaking).
- **Ton** : ✅✅✅ (le « soir d'élection »).
- **Compat. maths** : ⚠️ enchère = flux d'argent + voix. **Vu dans : A, C.**

**Escalade de fin de partie (« Dernière ligne droite »)** *(King of Tokyo)*
- **Description** : quand un joueur dépasse un palier (ex. 18 sur la piste), on retourne une carte qui accélère la fin : revenus +5/tour, votants restants +1 voix, dénonciations reculent de 2 au lieu de 1.
- **Ressort addictif** : accélération finale → comeback possible + tension, anti-fin molle.
- **Remplace / restructure** : ajout de fin de partie.
- **Coût / équilibrage** : faible. Risque faible.
- **Ton** : ✅ (« dernière semaine de campagne »).
- **Compat. maths** : ⚠️ modifie voix/argent en fin. **Vu dans : A.**

## C. Refonte de l'attaque (dénonciation / vol)

**Commission d'Enquête Législative** *(Sheriff of Nottingham, John Company)*
- **Description** : on **supprime** la dénonciation sèche. À la place, 1 action + 2 M€ ouvrent une enquête sur un front adverse → **débat de 60 s** : les tiers misent (roulette) pour soutenir l'accusation OU financer la défense (et toucher une part en cas de non-lieu) ; l'accusé négocie, corrompt, ou plaide coupable. Résolution = révélation de la carte ciblée → coupable : perte de voix + transferts d'argent ; non-lieu : amende pour diffamation aux accusateurs.
- **Ressort addictif** : théâtre social + négociation asymétrique sous chrono + *near-miss* (l'accusateur risque une amende).
- **Remplace / restructure** : remplace la **dénonciation 2 M€** (cœur de l'attaque).
- **Coût / équilibrage** : moyen ; calibrer les récompenses pour ne pas asphyxier l'accusé. Risque moyen.
- **Ton** : exceptionnel (théâtralisation judiciaire).
- **Compat. maths** : ⚠️⚠️ change l'attaque centrale + flux d'argent. **Vu dans : C (reco #1).**

**OPA hostile / Scission partisane** *(Monopoly Deal)*
- **Description** : racheter un bloc déjà acquis par un rival, à condition qu'il soit **incompatible** avec son électorat ; offre publique = **2× la valeur faciale** ; le défenseur accepte l'argent ou bloque par une carte d'action.
- **Ressort addictif** : agression directe, déconstruction du moteur de score adverse.
- **Remplace / restructure** : renforce notre vol d'électorat standard.
- **Coût / équilibrage** : élevé ; risque de frustration destructrice → compenser financièrement la cible. Risque élevé.
- **Ton** : élevé (ralliements opportunistes).
- **Compat. maths** : ⚠️ gros swing de voix. **Vu dans : C.**

**Tarification adaptative des blocs (rétroaction négative)** *(Power Grid)*
- **Description** : le coût d'un bloc de votants dépend de ta **position sur la piste** : le meneur paie plus cher, le retardataire moins.
- **Ressort addictif** : anti-*runaway leader* passif (sans punir socialement) → la course reste serrée → on rejoue.
- **Remplace / restructure** : ajuste le coût fixe des votants.
- **Coût / équilibrage** : moyen. Risque moyen.
- **Ton** : ✅ (« plus t'es haut, plus ça coûte de corrompre »).
- **Compat. maths** : ⚠️ touche l'argent + l'équilibre. **Vu dans : C.**

## D. Les défenses & la corruption

**Push-your-luck sur la corruption (« Financement occulte / Casino offshore »)** *(Quacks, Incan Gold, Can't Stop)*
- **Description** : option « quitte ou double ». Variante revenu : renoncer aux 3 M€ légaux pour piocher des Combines une à une (+3 M€/corruption révélée), mais 2 corruptions du même front = échec (tout perdu + poser une carte sale). Variante voix : « Casino Offshore » double la mise → pile/face → succès = grosse prime de voix, échec = perdre la moitié de l'argent.
- **Ressort addictif** : frisson du pari, dopamine du « encore une carte », illusion du coup de poker.
- **Remplace / restructure** : alternative au revenu passif / à la corruption (nouvelle action optionnelle ou carte « Investisseur mystère »).
- **Coût / équilibrage** : faible (un marqueur/carte). Risque moyen (runaway par chance).
- **Ton** : bon (gambling politique, sondages truqués, casinos offshore).
- **Compat. maths** : ⚠️ touche surtout l'**argent** (zone que tu voulais préserver). **Vu dans : A, B, C (la seule citée par les 3).**

**Taxe d'Hypocrisie** *(Suburbia)*
- **Description** : au lieu d'un malus unique de −1 voix, chaque incompatibilité active dans ta coalition génère une dette récurrente : à la phase de revenu, tu paies **1 M€/tour par incompat** pour maintenir la cohésion.
- **Ressort addictif** : dilemme éco à moyen terme (coalition bancale vs moteur de revenus).
- **Remplace / restructure** : remplace l'incompatibilité molle −1 voix.
- **Coût / équilibrage** : moyen (revoir la vitesse de circulation de l'argent). Risque moyen.
- **Ton** : excellent (s'asphyxier pour des promesses contradictoires).
- **Compat. maths** : ⚠️ touche l'**argent** + l'incompat calibrée. **Vu dans : C.**

**Blanchiment offshore à retardement** *(dilemme du prisonnier)*
- **Description** : le blanchiment ne s'efface plus tout de suite ; l'argent va dans un **compte offshore collectif** bloqué. En fin de manche, si le total dépasse un seuil → audit mondial qui révèle les cartes les plus sales de chaque contributeur.
- **Ressort addictif** : risque partagé, tension collective.
- **Remplace / restructure** : remplace le blanchiment immédiat (3 M€).
- **Coût / équilibrage** : faible (une zone de dépôt). Risque faible.
- **Ton** : élevé (paradis fiscaux).
- **Compat. maths** : ⚠️ change le timing de la défense. **Vu dans : C.**

**Contestation judiciaire / bluff de procédure** *(Coup)*
- **Description** : l'accusé oppose une défense face cachée (« c'est le bon front ») ; l'attaquant peut **contester**. Bluff démasqué = malus **doublé** ; contestation à tort = l'attaquant perd son action + **amende 5 M€**.
- **Ressort addictif** : bluff pur, intimidation, paranoïa.
- **Remplace / restructure** : transforme « Élément de langage » réactif en jeu d'esprit actif.
- **Coût / équilibrage** : moyen. Risque élevé (calibrer les pénalités).
- **Ton** : élevé (avocats d'affaires, vices de procédure).
- **Compat. maths** : ⚠️ amendes (argent). **Vu dans : C.**

**Fuite médiatique ciblée** *(original)*
- **Description** : si tu accumules plus de 3 cartes de corruption sur un même front, la carte du dessus est **révélée face visible** à toute la table (les dénonciateurs connaissent la valeur exacte).
- **Ressort addictif** : gestion de l'exposition, paranoïa, punit la gourmandise.
- **Remplace / restructure** : ajuste la règle de visibilité des fronts.
- **Coût / équilibrage** : très faible. Risque très faible.
- **Ton** : ✅ (plus le scandale est gros, plus ça fuite).
- **Compat. maths** : ✅ (info, n'ajoute ni voix ni argent). **Vu dans : C.**

## E. L'asymétrie & les rôles

**Pouvoirs asymétriques renforcés** *(Root, Cosmic Encounter, Scythe)*
- **Description** : amplifier les 6 partis avec une capacité unique forte (1×/partie ou continue). Ex. populiste double une « Promesse intenable » une fois ; un parti « Surveillés » blanchit gratuitement 1 corruption/tour ; conservateur décrète un bonus de protection.
- **Ressort addictif** : chaque faction crée un style de jeu inédit → découverte + rejouabilité.
- **Remplace / restructure** : se greffe sur l'existant (ne pas trop complexifier).
- **Coût / équilibrage** : modéré ; risque moyen-élevé si un pouvoir domine.
- **Ton** : excellent (satire de privilèges/slogans).
- **Compat. maths** : ⚠️ chaque pouvoir touche voix/argent → re-sim cas par cas. **Vu dans : A, B, C.**

**Conseiller drafté chaque manche** *(Citadelles)*
- **Description** : en plus de ton parti fixe, tu draftes un « Conseiller » dans une rivière de 6 (Spin doctor : +1 dénonciation ; Trésorier : +2 M€ ; Lobbyiste : 1er votant −2 M€…). Tu en prends 1, tu passes le reste.
- **Ressort addictif** : micro-décision fraîche chaque manche + ciblage (prendre le rôle convoité).
- **Remplace / restructure** : dynamise les 6 partis figés sans les supprimer.
- **Coût / équilibrage** : faible-moyen. Risque faible.
- **Ton** : ✅✅ (galerie de seconds couteaux).
- **Compat. maths** : ⚠️ bonus → re-sim léger. **Vu dans : A.**

## F. La négociation & le social

**Alliances temporaires / Cabinets de coalition** *(Cosmic Encounter, Diplomacy, Chinatown)*
- **Description** : deux joueurs scellent un pacte (pion « Cabinet commun ») pour 1-2 manches : ils valident un bonus de coalition fort et partagent librement argent/voix. Mais l'un peut **trahir** à son tour pour rafler tout le bonus, au prix d'une carte de corruption « Rue » (coût politique).
- **Ressort addictif** : la trahison = la meilleure narration émergente connue (Diplomacy). On rejoue pour le drama.
- **Remplace / restructure** : remplace/enrichit « Renvoi d'ascenseur » (jugé trop statique).
- **Coût / équilibrage** : moyen ; risque kingmaking (incitations fortes à trahir pour éviter une alliance qui bloque la table).
- **Ton** : ✅✅✅ (accords de gouvernement, trahisons partisanes).
- **Compat. maths** : ⚠️ partage de voix/argent. **Vu dans : A, C.**

**Négociation libre totale** *(Chinatown)*
- **Description** : libéraliser tous les actifs : argent de la roulette, blocs d'électeurs et cartes des Combines deviennent négociables à tout moment. On achète le silence d'un rival, on cofinance un bloc, on vend des promesses de non-agression.
- **Ressort addictif** : espace social pur, alliances éphémères, spéculation sur l'avenir.
- **Remplace / restructure** : étend la « négociation libre » déjà permise par les règles.
- **Coût / équilibrage** : faible (c'est une autorisation). Risque moyen (kingmaking, analysis paralysis).
- **Ton** : ✅✅✅.
- **Compat. maths** : ✅ conceptuel (déjà autorisé au chat). **Vu dans : C.**

**Vote législatif** *(A Game of Thrones, John Company)*
- **Description** : périodiquement, un joueur propose une « loi » (règle temporaire) votée ensemble ou acceptée contre un pot-de-vin. Ex. « les achats de votants coûtent +1 M€ ce tour », « chacun doit payer 2 M€ pour gagner 2 voix ».
- **Ressort addictif** : pouvoir social direct + drama (débats, marchandages).
- **Remplace / restructure** : introduit un vrai système de négociation politique (peut influencer taxes/seuils).
- **Coût / équilibrage** : élevé en règles (allonge le jeu, AP) → à utiliser avec parcimonie.
- **Ton** : humoristique (lois absurdes).
- **Compat. maths** : ⚠️ change des constantes en cours de partie. **Vu dans : B, C.**

**Enchères d'ordre du tour** *(Modern Art, Power Grid)*
- **Description** : avant chaque manche, mini-enchère pour choisir l'ordre de jeu ou un privilège temporaire (le plus offrant devient « présentateur principal »).
- **Ressort addictif** : interaction par pari immédiate (« qui paie pour agir avant les autres ? »).
- **Remplace / restructure** : remplace le jeton premier joueur tournant (§7).
- **Coût / équilibrage** : moyen ; limiter l'inflation/l'avantage. Risque moyen.
- **Ton** : satirique (acheter l'initiative, le média).
- **Compat. maths** : ⚠️ flux d'argent. **Vu dans : B.**

## G. Modules de variance

**Cartes d'Actualité / Événements / Scandale d'État** *(Pandemic, Dead of Winter, Root, John Company)*
- **Description** : un deck commun d'événements politiques pioché régulièrement (chaque manche, ou quand un seuil/une manche sans dénonciation passe). Ex. « Crise économique : revenus −1 », « Manifestation : chacun perd un votant aléatoire », « Scandale médiatique : non protégés reculent », « Débat télévisé : dernier −1, premier +1 », « Lanceur d'alerte / hacker russe », « Grève générale contre la corruption ». Peut modifier les règles de la manche suivante.
- **Ressort addictif** : imprévu + narration émergente (la table réagit à la crise du moment) → brise les routines.
- **Remplace / restructure** : réintègre — de façon structurée — le module d'événements retiré en v0.3 (§7).
- **Coût / équilibrage** : faible (~10-20 cartes), modérer l'impact. Risque moyen si trop punitif.
- **Ton** : élevé (satire de l'actu réelle).
- **Compat. maths** : ⚠️ effets sur voix/argent. **Vu dans : B, C.**

**Variance de setup (« lois électorales » / constantes tirées)** *(Root, Gloomhaven)*
- **Description** : tirer 3 « lois électorales » en début de partie qui modifient les constantes pour toute la partie (« protections à 3 M€ », « les Retraités valent 9 »…).
- **Ressort addictif** : aucune partie identique → rejouabilité par recombinaison.
- **Remplace / restructure** : ajout de setup.
- **Coût / équilibrage** : moyen-élevé (équilibrage difficile, beaucoup de combinaisons).
- **Ton** : ✅✅.
- **Compat. maths** : ⚠️⚠️ change les valeurs calibrées par partie. **Vu dans : A.**

## H. Méta / écarté

**Méta-progression cosmétique** *(roguelite, en ligne)*
- **Description** : en ligne uniquement, débloquer de nouveaux blocs/scandales/skins au fil des parties (cosmétique + variantes).
- **Ressort addictif** : carotte de déblocage = rétention numérique classique.
- **Remplace / restructure** : couche en ligne, ne touche pas au jeu de base.
- **Coût / équilibrage** : moyen (dev), nul en équilibrage si purement cosmétique.
- **Ton** : ✅.
- **Compat. maths** : ✅. **Vu dans : A, B.**

**Campagne / Legacy** *(Risk Legacy, Pandemic Legacy)* — ⚠️ **DÉCONSEILLÉ par les 3 sources**
- **Description** : lier plusieurs parties (progression permanente, contenu débloqué, altération du matériel).
- **Ressort addictif** : continuité narrative.
- **Remplace / restructure** : casserait la nature de partie isolée.
- **Coût / équilibrage** : très élevé ; déséquilibre vétérans/nouveaux ; alourdit.
- **Ton** : risque de trancher avec la satire ouverte.
- **Compat. maths** : ❌ casse le format « relance rapide ». **Vu dans : A, B (rejeté), C (anti-piège).**
